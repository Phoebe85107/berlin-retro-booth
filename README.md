A fully immersive, web-based vintage photo booth application inspired by the iconic 80s-90s chemical photo booths of Berlin.

## ‚ú® Features

- **Mechanical Curtain Animation**: Immersive entry experience.
- **Retro Filters**: Multiple 80s/90s inspired filters including Berlin B&W, Sepia, and Cyanotype.
- **Dynamic Photo Strip**: Generates a classic vertical 4-frame photo strip.
- **Animated Strip (MP4/WebM)**: Download an animated video of your session.
- **Mobile Optimized**: Clean grid layout for style selection on small screens.
- **Control System**: Pause or Abort the session at any time.

## üöÄ Quick Start (Local)

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Phoebe85107/berlin-retro-booth.git
   cd berlin-retro-booth
   ```

2. **Run locally:**
   Since this project uses ESM modules and Import Maps, you can simply serve the directory:
   ```bash
   # Using npx
   npx serve .
   ```

3. **Open in Browser:**
   Visit `http://localhost:3000` (or the port provided by serve).
