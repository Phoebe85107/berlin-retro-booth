- **Mechanical Curtain Animation**: Immersive entry experience with CSS-driven curtain mechanics.
- **Retro Filters**: Multiple 80s/90s inspired filters including Berlin B&W, Cyanotype, and Fujifilm NC style.
- **Dynamic Photo Strip**: Generates a classic vertical 4-frame photo strip with analog noise and paper textures.

- **Responsive Design**: Works on mobile and desktop, mimicking the intimate space of a real booth.
- **Hardware Controls**: Toggle torch/flash and mirror view directly from the UI.

## üöÄ Deployment

The app is built using ESM modules and requires no complex build step. To deploy:

1. Push to GitHub.
2. Connect your repository to **Vercel** or **Netlify**.
3. Ensure **HTTPS** is enabled (required for camera access).
